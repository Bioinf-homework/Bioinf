## 安装依赖 

### fedora 22+

```
sudo dnf install gcc-c++
sudo dnf install shedskin
sudo dnf install wxPython
sudo pip install matplotlib
```

### ubuntu
```
sudo apt-get install g++
sudo apt-get install shedskin
sudo pip install matplotlib
sudo pip install wx
```

### 编译脚本准备
chmod +x setup.sh
./setup.sh
